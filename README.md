# TODOLIST PYTHON
To-Do List using Python is a simple console-based application that allows users to add, view, and manage their daily tasks efficiently. The program stores tasks in a list, provides a menu-driven interface for adding and displaying tasks, and continues running until the user chooses to exit. This project demonstrates the use of Python concepts such as lists, loops, conditional statements, user input, and basic list operations.

# EXPENSE TRACKER USING PYTHON
Expense Tracker using Python is a simple console-based application that allows users to enter multiple expense amounts and calculates the total amount spent. The program continuously accepts user input until the user types done, validates the entered values using exception handling, and displays the final total expense. This project demonstrates the use of Python concepts such as variables, loops, conditional statements, user input, arithmetic operations, and exception handling.
